Basic Filtering with WHERE Clause

IRON

Given the STUDENTS table, show all students from "Delhi".

CREATE TABLE STUDENTS ( StudentID INT PRIMARY KEY, Name VARCHAR(50), Age INT, Grade CHAR(1), City VARCHAR(50), Marks INT ); INSERT INTO STUDENTS VALUES (1, 'Amit', 17, 'A', 'Delhi', 85), (2, 'Neha', 18, 'B', 'Mumbai', 72), (3, 'Raj', 17, 'A', 'Delhi', 90), (4, 'Simran', 19, 'C', 'Bangalore', 65), (5, 'Karan', 18, 'B', 'Kolkata', 78);

Solution

SELECT * FROM STUDENTS WHERE City = 'Delhi';

Result

StudentID Name Age Grade City Marks
1 Amit 17 A Delhi 85
3 Raj 17 A Delhi 90

Basic Aggregates with GROUP BY

BRONZE

Given the EMPLOYEES table, count the number of employees in each department.

CREATE TABLE EMPLOYEES ( EmpID INT PRIMARY KEY, Name VARCHAR(50), Department VARCHAR(50), Salary INT, JoinYear INT ); INSERT INTO EMPLOYEES VALUES (1, 'Amit', 'HR', 50000, 2018), (2, 'Neha', 'IT', 70000, 2019), (3, 'Raj', 'Finance', 60000, 2020), (4, 'Simran', 'IT', 75000, 2017), (5, 'Karan', 'HR', 52000, 2022);

Solution

SELECT Department, COUNT(*) as Emp_Count FROM EMPLOYEES GROUP BY Department;

Result

Department Emp_Count
HR 2
IT 2
Finance 1

Advanced Aggregates

SILVER

Given the ORDERS table, find the second highest price in orders.

CREATE TABLE ORDERS ( OrderID INT PRIMARY KEY, CustomerName VARCHAR(50), Product VARCHAR(50), Price INT, Quantity INT, OrderDate DATE ); INSERT INTO ORDERS VALUES (101, 'Amit', 'Laptop', 50000, 1, '2024-01-15'), (102, 'Neha', 'Phone', 30000, 2, '2024-02-10'), (103, 'Raj', 'Tablet', 20000, 1, '2024-03-05'), (104, 'Simran', 'Laptop', 55000, 1, '2024-03-20'), (105, 'Karan', 'Phone', 32000, 1, '2024-04-10');

Solution

SELECT MAX(Price) as Second_Highest_Price FROM ORDERS WHERE Price < (SELECT MAX(Price) FROM ORDERS);

Result

Second_Highest_Price
50000

Subqueries & Conditional Aggregation

GOLD

Given the DEPARTMENTS table, find the department with the highest budget.

CREATE TABLE DEPARTMENTS ( DeptID INT PRIMARY KEY, DeptName VARCHAR(50), Manager VARCHAR(50), Budget INT ); INSERT INTO DEPARTMENTS VALUES (1, 'HR', 'Amit', 500000), (2, 'IT', 'Neha', 800000), (3, 'Finance', 'Raj', 600000), (4, 'Marketing', 'Simran', 700000);

Solution

SELECT * FROM DEPARTMENTS ORDER BY Budget DESC LIMIT 1;

Result

DeptID DeptName Manager Budget
2 IT Neha 800000

Nested Subqueries & Rank Functions

MASTER

Given the EMPLOYEES table, find employees who earn more than their department's average salary.

CREATE TABLE EMPLOYEES ( EmpID INT PRIMARY KEY, Name VARCHAR(50), Department VARCHAR(50), Salary INT, JoinYear INT ); INSERT INTO EMPLOYEES VALUES (1, 'Amit', 'HR', 50000, 2018), (2, 'Neha', 'IT', 70000, 2019), (3, 'Raj', 'Finance', 60000, 2020), (4, 'Simran', 'IT', 75000, 2017), (5, 'Karan', 'HR', 52000, 2022), (6, 'Priya', 'Finance', 60000, 2021), (7, 'Rahul', 'HR', 75000, 2016), (8, 'Sneha', 'IT', 68000, 2018), (9, 'Manoj', 'Finance', 54000, 2017);

Solution

SELECT e1.* FROM EMPLOYEES e1 WHERE Salary > ( SELECT AVG(Salary) FROM EMPLOYEES e2 WHERE e2.Department = e1.Department );

Result

EmpID Name Department Salary JoinYear
3 Raj Finance 60000 2020
4 Simran IT 75000 2017
6 Priya Finance 60000 2021
7 Rahul HR 75000 2016
